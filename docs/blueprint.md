# **App Name**: Accessible Hire

## Core Features:

- Accessible UI Controls: Implement essential accessibility features including a high contrast mode, adjustable font sizing, full keyboard navigation, and screen reader optimization (with ARIA labels) for an inclusive user experience.
- AI Interview Mode Selection: Allow candidates to choose their preferred interview format (text, voice, or sign language) through interactive, descriptive cards with clear icons.
- AI Text Interview Module: Enable text-based interviews using a chat-like interface where an AI tool dynamically interacts with the candidate, interpreting responses and generating follow-up questions.
- AI Voice Interview Module: Provide a voice-based interview experience where an AI tool processes spoken language via speech-to-text, interprets responses, and delivers spoken prompts or questions.
- Sign Language Interview Module: Implement a dedicated camera-based interface for sign language interaction during interviews, serving as a placeholder for future AI interpretation within an MVP.
- AI Assessment Question Generation: Dynamically generate inclusive skill assessment questions using a Gemini AI tool, adapting to diverse formats (text, voice, visual) for varied cognitive needs.
- Multi-Format Assessment Input: Allow candidates to provide assessment answers via text input, voice recordings, or interaction with visual-based prompts, incorporating options for extra time and adaptive difficulty messaging.

## Style Guidelines:

- Primary color: A deep, rich purple (#5128A3) to convey trustworthiness and innovation, contrasting effectively against lighter elements for accessibility.
- Background color: A very light, desaturated lilac (#F2F1F3), offering a clean, friendly, and soft base that subtly harmonizes with the primary color while ensuring excellent readability.
- Accent color: A vibrant, clear blue (#306CE8), chosen to create high contrast for interactive elements, call-to-action buttons, and key highlights, enhancing user engagement and clarity.
- Headlines and Body text: 'Inter', a modern grotesque-style sans-serif, ensuring optimal legibility, neutrality, and a clean, objective aesthetic for all content.
- Modern, clear, and easily distinguishable icons for features, especially for interview modes (text, voice, sign language), ensuring immediate comprehension for all users. Icons should maintain accessible contrast ratios.
- A minimalistic and clean layout with ample whitespace and generous rounded corners (16px+ border-radius) for cards and buttons. Emphasize a logical information hierarchy, prominent call-to-action elements, and intuitive navigation for all users.
- Smooth, subtle transitions and micro-interactions for state changes, form submissions, and mode toggles, enhancing user feedback and a polished feel without causing distraction or accessibility issues.
